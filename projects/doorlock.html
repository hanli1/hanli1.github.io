Overview

This is the final project in ECE 3140: Embedded Systems. Implementation of a password protect doorlock on the ARM based Freescale FRDM Board MK64U.

The parts written include the servo controls using PWM, or Pulse Width Modulation which allows us to control the direction the servo is currently pointing in.

In addition, we used a 4x3 keypad from Adafruit, which we combined with interrupts on Port C of the board to generate the appropriate column clicked. We then poll upon interrupt to determine which row is pressed. Using this we then determine from a matrix which key is actually pressed. This is linked to the software finite state machine.

Communication with the user is done with PUTTY client which connects directly to the board with USB.

Features
  • Simulation of a doorlock with servo control
  • Setting a unlimited length password with confirmation using a 4x3 physical keypad
  • Reset of existing password
  • Lockout of the user after multiple incorrect attempts
