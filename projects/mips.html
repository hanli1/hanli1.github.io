Overview

Implementation of a 8-bit MIPS style microprocessor with branching and halting in a hardware description language to run on the FPGA Cyclone 5.

Features
  • 8-bit ALU built from scratch
  • Branching and halting instructions
  • All other common instructions to enable us to write a heart rate monitor that lights up LEDs based on how fast a person is clicking a button
Languages Used

Verilog
